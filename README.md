📊 Sales Performance Dashboard (Power BI)
🚀 Project Overview

This project is an interactive Sales Performance Dashboard built using Power BI to analyze retail sales data and generate actionable business insights.

The dashboard enables decision-makers to monitor performance, identify trends, and optimize sales strategies using data-driven insights.

🎯 Objectives
Analyze overall sales and profit performance
Identify top-performing regions, categories, and products
Track monthly and yearly sales trends
Detect loss-making segments and improvement areas
📂 Dataset
Source: SuperStore Sales Dataset
Format: Excel (.xlsx)
Contains:
Orders data
Customer details
Product categories
Sales, profit, and quantity metrics
🛠️ Tools & Technologies
Power BI Desktop – Dashboard creation & visualization
Microsoft Excel – Data storage
Power Query – Data cleaning & transformation
DAX (Data Analysis Expressions) – Calculations & measures

📈 Key Features
📅 Time-based sales analysis (monthly/yearly trends)
🌍 Region and segment-wise performance tracking
📦 Category & sub-category breakdown
💰 Profit vs Sales comparison
🎛️ Interactive slicers for dynamic filtering
🔍 Key Insights
Certain categories generate high sales but low profit, indicating pricing or cost issues
Western region shows strong performance, while some regions need improvement
Seasonal trends highlight peak sales periods, useful for planning campaigns
Discounts significantly impact profit margins
💼 Business Value

This dashboard helps:

Improve decision-making using real-time insights
Identify profitable and loss-making areas
Optimize inventory and marketing strategies
Enhance overall business performance
